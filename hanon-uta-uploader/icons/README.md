# Icons

This directory should contain the following icon files:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## How to Create Icons

### Option 1: Use Online Tools
1. Visit https://www.favicon-generator.org/ or similar
2. Upload an image or create a design
3. Download in multiple sizes

### Option 2: Use Image Editor
1. Create a simple design (e.g., a microphone icon)
2. Export in three sizes: 16x16, 48x48, 128x128
3. Save as PNG format

### Option 3: Use Placeholder (Temporary)
For testing, you can use any PNG images with the correct dimensions.

## Recommended Design

A simple microphone icon with the following colors:
- Background: #2196F3 (blue)
- Icon: white
- Style: flat or minimal

## Temporary Solution

For immediate testing, you can:
1. Download any 16x16, 48x48, and 128x128 PNG images
2. Rename them to icon16.png, icon48.png, icon128.png
3. Place them in this directory

The extension will work with any images as long as they have the correct filenames and dimensions.
